# 初筛与证据表建议

## 标题/摘要初筛（Title/Abstract Screening）

目标：用一致的规则把“可能相关”与“明显不相关”分开，降低后续全文筛选成本。

- 纳入（常见）：与 ASD/ADHD/neurodiversity 直接相关；人类研究；有明确方法与样本；与你的目标人群/情境匹配
- 排除（常见）：纯动物/细胞/基础机制且与你的综述目标无关；只提到关键词但主题不相关；重复发表/明显重叠
- 记录原因：每篇排除给 1 个最主要原因（便于复核）

## 证据表字段（建议 CSV/Markdown）

最小字段集（够用且可扩展）：

- Citation：作者（年份）标题，期刊
- PMID / DOI / URL
- Study type：systematic review / meta-analysis / RCT / cohort / cross-sectional / qualitative
- Population：年龄段、样本量、诊断标准、地区/语言
- Exposure/Topic：你关注的变量（例如 masking、executive function、employment support）
- Key findings：一句话结论 + 可引用数据点
- Limitations：主要局限（偏倚、样本、测量、外推性）
- Notes：与你写作框架的对应位置（哪一节/哪一个论点）

## 表达与框架提醒（Neurodiversity-affirming）

- 区分“障碍模型”与“神经多样性框架”的讨论语境，避免把差异默认等同于缺陷
- 对于功能与支持需求的描述，优先使用中性、具体、可操作的语言（避免泛化与刻板化）
